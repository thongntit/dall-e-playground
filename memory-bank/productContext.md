# Product Context

## Project Overview
DALL-E Playground is an unofficial web application providing a user-friendly interface for OpenAI's DALL-E image generation APIs. The application supports both DALL-E 2 and DALL-E 3 models, enabling users to generate, modify, and manipulate AI-created images through natural language prompts.

## Core Features
- Support for DALL-E 2 and DALL-E 3 models
- Multi-variant image generation (DALL-E 2)
- Chat-based interface
- Local storage for images and chat history
- Image imprinting capability
- Responsive design
- Error handling
- Generation control

## Technical Stack
- Frontend: React 18 with TypeScript
- Styling: TailwindCSS
- State Management: Zustand
- Routing: React Router
- API: OpenAI
- Build Tool: Vite
- Storage: LocalForage
- UI Components: Radix UI primitives

## Project Architecture
The application follows a modern React architecture with clean separation of concerns:
- Components for UI elements
- Stores for state management
- Hooks for shared logic
- Types for TypeScript definitions
- Routes for page management
- Utils for shared utilities

## Integration Points
- OpenAI API for image generation
- LocalForage for persistent storage
- Browser APIs for image handling
