# Changelog

## [1.0.0] - 2025-04-02

### Initial Release

We're excited to announce the first official release of DALL·E Playground with Imprint Image! This application allows you to interact with OpenAI's image generation models in an intuitive chat interface.

### Features

- **Multi-Model Support**: Works with both DALL·E 3 and DALL·E 2 models
- **Multiple Variants**: Generate multiple image variations (DALL·E 2 only)
- **Chat Interface**: Intuitive chat-based UI for generating images
- **Image Imprinting (MVP)**: Edit existing generated images by selecting areas to modify with new prompts (DALL·E 2 only)
- **Persistent Storage**: All your conversations and generated images are saved locally
- **Responsive Design**: Works on various screen sizes
- **Generation Controls**: Easily cancel ongoing image generations
- **Error Handling**: Robust error feedback for API issues
- **Configuration Options**: Control model settings like style, size, and quality

### Requirements

- You must use your own [OpenAI API Key](https://platform.openai.com/account/api-keys)
