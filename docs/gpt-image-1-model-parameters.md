# GPT-Image-1 Model Parameters

This document details the specific parameters and capabilities of OpenAI's "gpt-image-1" model based on the OpenAI SDK type definitions.

## Basic Information

- **Model ID:** `gpt-image-1`
- **Supported Operations:** Image generation and image editing

## Parameters Comparison with DALL-E Models

| Parameter | gpt-image-1 | DALL-E 3 | DALL-E 2 |
|-----------|-------------|----------|----------|
| Size options | 1024x1024, 1792x1024 (landscape), 1024x1536 (portrait), auto (default) | 1024x1024, 1792x1024, 1024x1792 | 256x256, 512x512, 1024x1024 |
| Quality options | high, medium, low | hd, standard | standard |
| Output format | png, jpeg, webp | N/A | N/A |
| Max prompt length | 32,000 characters | 4,000 characters | 1,000 characters |
| Response format | Always returns base64-encoded images | url or b64_json | url or b64_json |

## Unique Parameters for gpt-image-1

### 1. Background Transparency
```typescript
/**
 * Allows to set transparency for the background of the generated image(s).
 * This parameter is only supported for `gpt-image-1`.
 * Must be one of `transparent`, `opaque` or `auto` (default value).
 */
background?: 'transparent' | 'opaque' | 'auto';
```

### 2. Content Moderation Level
```typescript
/**
 * Control the content-moderation level for images generated by `gpt-image-1`.
 * Must be either `low` for less restrictive filtering or `auto` (default value).
 */
content_moderation?: 'low' | 'auto';
```

### 3. Compression Level
```typescript
/**
 * The compression level (0-100%) for the generated images.
 * This parameter is only supported for `gpt-image-1` with the `webp` or `jpeg` output formats.
 * Defaults to 100.
 */
compression?: number;
```

### 4. Format
```typescript
/**
 * The format in which the generated images are returned.
 * This parameter is only supported for `gpt-image-1`.
 * Must be one of `png`, `jpeg`, or `webp`.
 */
format?: 'png' | 'jpeg' | 'webp';
```

## Response Differences

- gpt-image-1 returns token usage information in responses:
  ```typescript
  /**
   * For `gpt-image-1` only, the token usage information for the image generation.
   */
  usage?: {
    prompt_tokens: number;
    total_tokens: number;
  };
  ```

## Image Editing Capabilities

- For gpt-image-1, image editing supports:
  - Multiple images can be provided (unlike DALL-E 2)
  - Supported formats: png, webp, or jpg files
  - Size limit: less than 25MB per image

## Implementation Considerations

1. **UI Updates Needed:**
   - Add new quality options (high, medium, low)
   - Add new format options (png, jpeg, webp)
   - Add transparency control
   - Add compression level control
   - Add content moderation level control

2. **API Parameter Handling:**
   - The application will need to conditionally include gpt-image-1 specific parameters
   - Response handling will need to account for token usage information

3. **Response Handling:**
   - Always expect base64-encoded images from gpt-image-1
   - Process token usage information when available

4. **Default Values:**
   - Size: 'auto'
   - Quality: 'auto'
   - Background: 'auto'
   - Content moderation: 'auto'
   - Format: 'png'
   - Compression: 100 (for webp/jpeg)
